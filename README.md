## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔐 **Auth** | Register, login, forgot password, JWT refresh |
| 🏠 **Dashboard** | Progress stats, streaks, course recommendations |
| 🐍 **Python Course** | 10 modules from basics to APIs |
| 🤖 **AI Course** | Generative AI, prompt engineering, LLM apps |
| 📊 **ML Course** | Data prep → model training → deployment |
| 💬 **AI Tutor** | Claude-powered chat tutor with markdown + code |
| 📝 **Quiz System** | MCQ, instant grading, pass/fail logic |
| 🏆 **Certificates** | Auto-generated PDF with QR verification |
| 📈 **Progress** | Per-lesson tracking, streaks, badges |
| ⚙️ **Admin Panel** | Users, courses, analytics dashboard |
| 🌙 **Dark / Light** | Full theme system with smooth transitions |

## 🚀 Quick Start

```bash
cp backend/.env.example backend/.env   # Add your ANTHROPIC_API_KEY
docker-compose up --build -d
```
See [docs/SETUP_GUIDE.md](docs/SETUP_GUIDE.md) for full instructions.

## 🛠 Tech Stack

**Backend:** Flask · SQLAlchemy · PostgreSQL · JWT · Anthropic Claude · ReportLab  
**Frontend:** Flutter · Dio · go_router · flutter_animate · fl_chart · flutter_markdown  
**DevOps:** Docker · Docker Compose · Nginx · Gunicorn
